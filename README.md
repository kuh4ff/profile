## 🛠️ Projects & Tools

### 🔍 Web Security Testing Suite
A collection of custom scripts and configurations for automated web security testing.

```
Technologies: Python, Bash, Docker
Targets:      SQLi, XSS, SSRF, IDOR, RCE, LFI
```

### 🌐 API Security Scanner
Modular API fuzzing framework for discovering endpoint vulnerabilities.

```
Technologies: Go, Python
Techniques:   JWT attacks, Mass Assignment, Rate limiting, GraphQL introspection
```

### 📡 Network Recon Toolkit
Automated network mapping and service enumeration scripts.

```
Technologies: Python, Nmap, Masscan
Features:     Port scanning, Service fingerprinting, Vulnerability detection
```

### 🧪 Lab Environments

